# Pre-Entrega_3_MoyanoManchon

## Orden:

1. Primero hacer las migraciones si es necesario: 'python manage.py makemigrations'.
2. Seguidamente: 'python manage.py migrate'.
3. Finalmente, para correr el servidor: 'manage.py'.
